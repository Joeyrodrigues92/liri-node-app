Language Interpretation and Recognition Interface (LIRI) with Node.js

#Description

Using the command line interface and Node, you can pass arguments and retrieve information from Spotify, Twitter, and OMDB.

#Technologies

* Node
* Node fs
* JavaScript
* Twitter NPM Package
* Spotify NPM Package
* Request NPM Package
* OMDB API
